# Branding

This directory contains trademark and brand identity documentation for KERI Auth.

## Contents

| File | Purpose |
|------|---------|
| [trademark-policy.md](trademark-policy.md) | Trademark usage policy for the "KERI Auth" name and logos |

## Brand Assets in Other Locations

For build simplicity, branded assets remain in their original locations within the Extension project:

- **Logos**: `Extension/wwwroot/images/` (logo.svg, logoB.svg, and sized PNGs)
- **Terms of Use**: `Extension/wwwroot/content/terms.html`
- **Privacy Policy**: `Extension/wwwroot/content/privacy.html`

## Runtime Source of Truth

The runtime brand constant is `AppConfig.ProductName` in `Extension/AppConfig.cs`.
