Release Notes - Beta version 0.3.1

DIGN Identity Wallet

Thank you for installing the Beta Release, now rebranded as DIGN Identity Wallet.

Limitations

Please note that this is an early version of the extension and as such may have some limitations and bugs compared to improved features and quality in future versions. As a beta release, it is not intended for production use. We are actively working to improve it, and appreciate your feedback.

Possible Bugs

As this is a testing phase, there may be undiscovered bugs that could potentially affect your experience, including the loss of data managed by the extension.

Feedback

We encourage you to provide feedback on any issues you encounter or suggestions for improvements. Please submit your feedback via aid@dign.id, or create a GitHub issue at https://github.com/KERIAuth/keriauth-blazor-wasm/issues.

Updates

We will update the extension to fix bugs and implement improvements. Updates may occur automatically if the software was installed via the Chrome Web Store or an enterprise deployment.
