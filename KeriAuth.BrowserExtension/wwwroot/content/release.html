Release Notes - Beta version 0.2.0

KERI Auth for Desktop Browsers

Thank you for installing the Beta Release!

Limitations

Please note that this is an early version of the extension and as such may have some limitations and bugs compared to improved features and quality in future versions. As a beta release, it is not intended for production use. We are actively working to improve it, and your feedback is invaluable to us.

Possible Bugs

As this is a testing phase, there may be undiscovered bugs that could potentially affect your experience, including the loss of data managed by the extension.

Data Collection

We do not intend to automatically collect any data, but only what you explicitly send us. For more details, please read our Privacy Policy, accessible from the menu.

Feedback

We encourage you to provide feedback on any issues you encounter or suggestions for improvements. Please submit your feedback via Discord or a GitHub issue at https://github.com/KERIAuth/keriauth-blazor-wasm/issues.

Updates

We will update the extension to fix bugs and implement improvements. Updates may occur automatically if installed via the Chrome Web Store.
